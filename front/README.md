# OWL 多智能体任务中心 · 可视化前端（Vue + Three.js）· v4

## 快速开始
```bash
npm i
npm run dev
```

## 打包
```bash
npm run build
```

## 特效清单
- 三维粒子主控立方体（流动/闪烁/脉冲反馈）
- 环形 Agent 粒子方块（外框+内部粒子）
- 主控 ↔ Agents 半透明蓝色连线（随场景自适应）
- “快递粒子”沿连线循环运动，表现任务派发
- **能量冲击波**：提交任务时从主控中心扩散
- 后处理 **Bloom 泛光**
- 星云粒子背景、OrbitControls 自动环绕
- 左侧 **历史追溯**，带 **展开/隐藏**，隐藏时主画布自适应放大
- 点击 Agent 查看实时日志
